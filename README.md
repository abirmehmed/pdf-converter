# 📄 PDF to PNG Converter

A fast, free, and browser-based PDF to PNG converter. No installation required!

## ✨ Features

- 🚀 Convert PDF pages to high-quality PNG images
- 🌐 Runs entirely in your browser (no server upload)
- 📥 Download individual pages
- 🎨 High-resolution output (2x scale)
- 🔒 100% private - your files never leave your computer

## 🚀 How to Use

1. Open `pdf-converter.html` in your browser
2. Click to select a PDF file
3. Wait for conversion
4. Download your PNG files!

## 🛠️ Technologies

- HTML5
- CSS3
- JavaScript (ES6+)
- [PDF.js](https://mozilla.github.io/pdf.js/) by Mozilla

## 📦 No Installation Needed

Just open the HTML file in any modern browser (Chrome, Firefox, Edge) and start converting!

## 🔐 Privacy

All processing happens locally in your browser. Your PDF files are **never** uploaded to any server.
